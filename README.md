# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Sairaj2033/Airbnb.git
```

```bash
cd Airbnb
```

---

## Install Dependencies

```bash
npm install
```

---

## Create Environment Variables

Create a `.env` file.

```env
ATLASDB_URL=

SECRET=

CLOUD_NAME=
CLOUD_API_KEY=
CLOUD_API_SECRET=

MAP_TILER=
```

---

## Start Development Server

```bash
npm start
```

Visit

```
http://localhost:8080
```
